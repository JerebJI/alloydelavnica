Vse je relacija

Vse je množica n-teric


						sig C {
							f : C
						}

f, C, 1 so relacije!

f je relacija {(C0,C1), (C0,C2), ...}

1 je relacija {(1)}

C je relacija {(C0), (C1), ...}

Torej operacije z relacijami delujejo "povsod"

univ množica vseh elementov
none prazna množica
iden relacija med elementi in samim sabo
+ unija množic (∪)
& presek množic (∩)
- odštevanje množic
= enakost množic
in relacija "je podmnožica" (⊆)
set potenčna množica
disj relacija disjunktnosti
^r tranzitivna ovojnica r
~r obrat relacije (x r y → y r x)
*r tranzitivno-refleksivna ovojnica relacij

Produkt relacij ->

  • Morda slika s puščicami
  • Primer s tabelami
  • daj primer array.index
  • ne pozabi števnost
  • ne pozabi, da je števna -> posebna
\[\begin{aligned}E[p \text{ -> } q]i = \{(&p_1,..,p_n, q_1,..,q_m) |\\ &(p_1,..,p_n) \in E[p]i\: \land\\ &(q_1,..,q_m) \in E[q]i\}\end{aligned}\]

A -> N B

vsak a se slika v N elementov B


							A -> one B -- funkcija

A M -> B

v vsak B se slika M elementov A


							A lone -> B -- injektivna relacija

A M -> N B

velja A M -> B in A -> N B


							A lone -> one B -- injektivna funkcija
							A one -> one B -- bijekcija

Kompozicija relacij .

  • Morda slika s puščicami
  • Primer s tabelami
  • daj primer array.index
\[\begin{aligned}E[p\: . q]i = \{(&p_1,..,p_{n-1}, q_2,..,q_m) |\\ &(p_1,..,p_n) \in E[p]i\: \land\\ &(q_1,..,q_m) \in E[q]i\: \land \\ &p_n = q_1\}\end{aligned}\]

Operator []

  • r.s = s[r]
  • t.(r.s) = r.s[t]
  • kaj z x[a,b]

Zožitev relacij <:, :>

  • Morda slika s puščicami
  • Primer s tabelami
  • daj primer array.index
\[\begin{aligned}E[p <: q]i = \{(&q_1,..,q_n) |\\ &q_1 \in E[p]i\: \land\\ &(q_1,..,q_n) \in E[q]i\}\end{aligned}\]

Povozi ++

  • relacija kot slovar, hashtable
p ++ q = p - (domena(q) <: p) + q
